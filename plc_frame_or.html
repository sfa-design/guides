Area

1. TOP Area

제일 상단에 배치되어 있으며 장비 네트워크 상태, 날짜, 일시, Admin, [장비명] 페이지명이 배치합을 기본 규칙으로 합니다.
우측에는 필수 고정버튼(Main, Admin, Buzzer, Alarm)과 언어선택 버튼이 배치되어야 합니다. 불필요시 Buzzer는 배치하지 않아도 됩니다.

2. Menu Are

대메뉴를 배치하는 영역으로 장비별로 명칭을 최대한 통일해서 사용해야합니다. 기본 상태에서는 Black-Gray 그라데이션 색상의 버튼을 사용하며,
활성화 상태에서는 흰색 바탕을 사용합니다.

3. Funtion Area

내용을 표시해주는 영역으로 자유롭게 배치하여 사용하도록 합니다. 상태를 표현하는 lamp 이외의 색상은 Gray 계열로 사용을 권장합니다.

3.1. Sub Menu Area : 대메뉴의 서브메뉴를 배치하는 영역으로 기본상태에서는 Gray 그라데이션 색상의 버튼을 사용하며 활성화상태에서는 파란색 버튼을 사용합니다.
3.2. ConTent Area : 내용을 배치하여 자유롭게 사용합니다. Sub Menu가 없을 시 Content 영역을 넓게 사용합니다.

4. Bottom Area

하단영역은 페이지에 상관받지 않는 버튼을 배치하거나 메세지박스를 배치하는 용도로 활용합니다. 다만 기본 색상은 변경하지 않도록 합니다.

TOP Area

Size

top영역 사이즈는 800px*50px로 top.png 이미지 파일을 변형없이 사용합니다.
600px*480px 핸디형 프로젝트의 경우 ()로 이미지파일을 변형없이 사용함을 기본으로 합니다.

Color

되도록 업로드된 이미지 사용을 권장합니다.
불가피한 상황시 색상값 #004080 (R0/ G64/ B128) 과 #002850 (R0/ G40/ B80) 값으로 그라데이션을 하여 사용해줍니다.
그라데이션 사용이 어려울 시 단색 #003264 (R0/ G50/ B100)을 사용합니다.

통신상태

PLC 통신상태를 좌측 상단에 표시합니다.
Online-회색, Offline-밝은연두색 이미지를 이용합니다.

Title

제목은 장비명과 페이지명을 적어줍니다.
요청이 있을 시에만 고객사 로고를 삽입하며 가능한 흰색로고를 사용합니다. 하단 다운로드 버튼에 없는 로고는 디자인 담당자 (현 박상화대리)에게 요청 부탁드립니다.

필수 고정버튼

TOP Area의 우측에는 필수 고정버튼이 배치되며 Main, Admin, Buzzer, System, Alarm이 이에 해당됩니다.

  • Main : 홈버튼 (필수)
  • Admin : 로그인 계정정보 (필수)
  • Buzzer : Buzzer ON/OFF (불필요시 삭제)
  • System : 장비 외 모든 설정
        (예) 언어, 화면, font 등 (불필요시 삭제)
  • Alarm : Alarm 페이지로 이동 (필수)
