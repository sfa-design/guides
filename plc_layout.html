Function

Function영역에는 레이아웃, 장비 제어 및 현황 등의 내용을 표현해 주는 영역입니다. Top영역과는 달리표현을 자유롭게 작성하여 사용합니다.
Submenu가 있을 시에만 좌측영역 사용해주며 해당내용은 업체 요청에 의해 변경가능한 권장사항입니다.
800*600이 표준 사이즈로 사용하며, Funtion영역의 색상은 되도록 회색계열로 전체적인 내용이 튀지않게 보이도록 배치하여 중요부분만 강조하여줍니다.
중요 부분이나 센서, 알람부분, 에러의 경우 채도가 강한 붉은색과 점멸을 사용하는 것을 권장합니다.

자주 사용하는 페이지 샘플은 아래 버튼에서 다운로드받아 사용하도록 합니다.

Function Detail

Submenu

대메뉴 하위메뉴인 Submenu가 존재 시 왼쪽에 배치하여줍니다.
짙은 회색박스 [사이즈 794px*464px, 배경색 #a0a0a0 / R160 G160 B160] 위에
내용부분의 흰색박스 669px*448px를 우측 여백을 두고 올려 사용합니다.
전체 Function영역에서 3px씩 여백을 주고, 다시 버튼이미지를 제외하고 8px의 여백을 주면 쉽게 계산 가능합니다.
Submenu버튼은 Enable 상태 : bt_submenu_en.png,
Press 상태 : 'bt_submenu_prs.png' 를 사용하여 표현해줍니다.

Table

레이아웃 내 색상은 현황을 나타내는 램프나 알람을 제외한 다른 내용은 회색계열을 사용하여 어느한 부분이 유난히 눈에 띄지 않게 배치합니다.
중요부분만 원색계열 (빨,노,파,녹)을 사용하여 강조합니다.
